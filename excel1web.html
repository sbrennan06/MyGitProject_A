Excel 1 Project Overview

The Excel 1 Project demonstrates the various excel functionalities which may be utilized to analyze raw data sets by filtering and sorting the data as well as creating data lookup tables, pivot tables charts, and graphs. This enhances data accessibility and the user's ability to compare and analyze that data quickly. These tools aid in business decision process, particularly the presentation of analysis.



As shown in the image below, the raw data is given as just text data in a table of cells. Some of the techniques utilized in the raw data worksheet are as follows:

  1. Column H has been conditionally formatted to highlight the cell green if and when the selling price is greater than $9.00.
  2. Data set was named "MusicData" as this allows for a VLookup table to be created.
  3. Text wrapping enabled on header to allow for full header text to be visible.


The primary excel functionality for which this project is focused is the VLookup functionality. This tool allows for associated data to be populated when a particular key or code is given. As shown in "VLookup Image 1" below, cell C5 is a fillable data field. When an existing code is entered, the remaining cells populate with the appropriate data. If that code does not exist, no data populates save for a message indicating "Code Not Found." A non-comprehensive list of the skills, techniques, and formulas used in the Lookup worksheet of this project are identified as follows:

  1. Header fields merged and centered.
  2. Date insertion tool utilized in merged and centered cell C3:K3 which automatically updates to today's date each time the file is opened.
  3. Customized cell background colors, font, and borders on each data section.
  4. Inserted Note in merged and centered cell B2:K2.
  5. Inserted arrow shape, customize color, size, border, and shape. The arrow is there to indicate the text entry cell C5, which is the only data entry field in the tab.
  6. VLookup function entered into cells C7:C12, C14:C15, and C18:C19. When an existing code is entered into cell C5, these cells will automatically populate with the corresponding data.
  7. There are three unique IFError functions utilized on this workbook that are used in tandem with the VLookup function either directly or indirectly. As shown above, when a code is populated into cell C7, the VLookup function will populate the corresponding data, however, if that code is invalid, the IFError part of the formula rules the data population of the cell.
      (a) In cell C7, if an error with the VLookup data exists, this cell will display "Code Not Found."
      (b) In cells C8:C12, C14:15, and C18:C19, if an error exists with the VLookup function, the IFError function will cause the display of blank cells.
      (c) Cells C16:C17 and C21:C22 are not subject to the VLookup formula, however, rely on the data of other cells which are subject to the VLookup function. If an error exists, the IFError function will cause the display of blank cells.
  8. The IF Function is used in cell C20 and C23 for unique purposes.
      (a) Cell C20 is not subject to a VLookup function but the IF Function is dependent on the results thereof. If a valid code is entered, cell C20 will display "15%" however, if the code entered was invalid resulting in cell C7 to display "Code Not Found," the IF Function will cause this cell to display no data.
      (b) An IF Function was installed in Cell C23 to provide a remark. If the value populated to cell C22 is equal to zero only, cell C23 will display the remark "The foal has been met, Yay!" Any other values will result in the display of a blank cell.
  9. A chart is inserted that automatically adjusts when a valid code is entered.
